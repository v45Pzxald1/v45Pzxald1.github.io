一、循环结构

循环结构是指在程序中需要反复执行某个功能而设置的一种程序结构。

举个例子

# include <bits/stdc++.h>
using namespace std;
int main() 
{
    for (int i = 1; i < 10; ++i)
    {
        printf("%d", i);
    }
    return 0; 
}

for语句

三种循环结构for、while和do-while，大多数情况它们是可以相互转换的。

for语句的运行逻辑

【例3.3】阅读以下代码，填写右侧表格。

# include <bits/stdc++.h>
using namespace std;
int main() 
{
    int i, j;
    for (i = 21, j = 5; i <= 50; ++i, j = j + 5)
    {
        if (i == j) printf("%d %d\n", i, j);
    }
    return 0; 
}
变量i的取值 变量j的取值 i 是否与 j 相等

提示： 建议尽量缩短变量的定义范围。例如，在for循环的初始化部分定义循环变量。

for语句的应用

【例3.4】给出一个正整数n\((1 \leq n \leq 10)\)，使用for语句完成以下内容。

  2. 计算\(n! = 1 \times 2 \times 3 \dots \times n\)。
  3. 计算\(\sum_{k=1}^n k! = 1! + 2! + \dots + n!\)
