一、循环结构

循环结构是指在程序中需要反复执行某个功能而设置的一种程序结构。

举个例子

# include <bits/stdc++.h>
using namespace std;
int main() 
{
    for (int i = 1; i < 10; ++i)
    {
        printf("%d", i);
    }
    return 0; 
}

四、多重循环

生活中的多重循环

  1. 年月日
  2. 时分秒

多重循环

举个例子

# include <bits/stdc++.h>
using namespace std;
int main() 
{
    for (int i = 1; i <= 3; ++i)
    {
        for (int j = 1; j <= 2; ++j)
        {
            printf("i:%dj:%d", i, j);
        }
    }
    return 0; 
}
