if语句格式

if语句格式：

if (表达式) 语句1;
else 语句2;

提示1：else 语句2;是可以省略的。

提示2：用花括号{}括起来，将几个语句括起来的语句组合称为复合语句，可以作为一个整体，充当if语句中的“语句1”或者“语句2”，且后面不需要加分号。

三、else if语句和嵌套if语句

else if语句

【例2.4】某社团有个小传统，每到夏天，老师都会给社团成员买雪糕解暑。今年打算从外卖平台订购，平台根据购买数量设置了明确的折扣规则，具体如下：

  • 购买 1~10 个：每个雪糕 2 元，无折扣；
  • 购买 11~30 个：每个雪糕按原价打 9 折；
  • 购买 31 个及以上：每个雪糕按原价打 8 折。

输入需要购买的雪糕数量，计算并输出应付的总钱数（结果保留 1 位小数）。

输入：35
输出：56.0

else if语句

【例2.4】

# include <bits/stdc++.h>
using namespace std;
int main() 
{
    double ans = 0.0;
    int num;
    scanf("%d", &num);
    if (num <= 10) ans = num * 2;
    else if (num > 10 && num <= 30) ans = num * 2 * 0.9;
    else ans = num * 2 * 0.8;
    printf("%.1lf", ans); 
    return 0; 
}

提示： 如果有多个并列、情况不交叉的条件需要一一处理，可以用else if语句。
